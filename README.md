# Comparative DNA Analysis Tool

## Overview
This project is a Python-based bioinformatics tool for analyzing DNA sequences.

## Features
- GC Content Calculation
- ORF Detection 
- DNA to Protein Translation
- Comparative Analysis of Sequences
- Visualization using Graphs

## Technologies Used
- Python
- Basic Bioinformatics Concepts
- Matplotlib

## How to Run
1. Run the Python file
2. Enter DNA sequences when prompted
3. View results and graph output

## Sample Output
- GC Content values
- ORF sequences
- Protein sequences
